# Daily-Task-Tracker
Daily Task Tracker – Google Sheet

This is a monthly task tracking system designed in Google Sheets.

On the left, I created a task list (Task 1–Task 10).
Across the top, I structured the full month (1–31 days) and divided it into 5 weeks for better analysis.
Each cell contains a checkbox to mark daily task completion.
Below the table, I used formulas to calculate:
Daily completed and incomplete tasks
Weekly performance summaries
On the right side, I added a progress section that compares goals vs actual completion.
At the bottom, I included donut charts to visually represent weekly progress and a monthly completion chart.
